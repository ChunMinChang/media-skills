
**Individual Bug Triage Helper**

- Analyses non-sec related issues in Firefox. Researches the current issue, finds duplicates, metas, and other bugs related to the issue.
- Provides background context which is incredibly valuable in speedy triage.
- Has the ability to dig into the code base and find fixes (not designed for this but still can.)
- Provides suggested next steps, including suggested correspondence with users.

- Contributions welcome!

See the **sample reports** sub directory for some examples.
